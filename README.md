# React E-Commerce Website (Demo Project)

This is a simple e-commerce website built using **React** (Create React App).  
It fetches product data from the [Fake Store API](https://fakestoreapi.com/products) and displays them in a modern UI with category filters and product details.

---

## Features

- 🛍️ **Product Listing** – Displays latest products with images, titles, and prices.
- 🧥 **Category Filters** – Quickly filter products by Men's Clothing, Women's Clothing, Jewelry, and Electronics.
- 🔗 **Product Details Page** – Each product links to its own page via React Router.
- 📦 **No Database Needed** – Uses Fake Store API as a mock backend, no real database required.
- ⚡ **Fast Development Setup** – Powered by Create React App (CRA).

---

## Tech Stack

- **Frontend:** React, React Router
- **Styling:** Bootstrap (or custom CSS)
- **API:** [Fake Store API](https://fakestoreapi.com/)
- **State Management:** React Hooks (`useState`, `useEffect`)
- **Others:** react-loading-skeleton (for loading effects)
